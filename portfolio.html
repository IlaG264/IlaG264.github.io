Welcome to my Portfolio!

Fish Aquarium

The Fish Aquarium is a simulation that has 3 moving fish and other aquatic sea life, with moving bubbles, simulating a aquarium.





Turtle Adventure



Turtle Adventure uses arrow keys to move the turtle throughout the deep sea, collecting different colored bubbles under a 20 second time limit. Each colored bubble is also customized to be a certain point value.







Obstacle Course



Obstacle Course is a scratch game where players choose between 3 different sprites in order to go through different backgrounds and avoid different obstacles like spikes and a lava floor.







A pHishy Fish Tank



A pHishy Fish Tank is a fish tank program that monitors alkalinity, pH, salinity, temperature, calcium, magnesium, and phosphate levels. There were issues with the alkalinity, temperature, and magnesium levels. We had to fix these errors in the code in order to make the program execute properly.
